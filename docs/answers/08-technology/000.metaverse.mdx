# 메타버스의 4가지 범주

## 1. 가상 세계와의 조우, 메타버스 개념
:::tip[개념]
가상 환경과 물리 환경을 융합한 온라인 공간에서 아바타를 이용해 사용자간 상호작용을 하는 가상 세계
